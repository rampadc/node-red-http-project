node-red-http-project
=====================

Example project with a single HTTP endpoint

Exposes a single endpoint `/things`.

```
GET /things
```

```
{
  "things": [
    "Raindrops on roses",
    "whiskers on kittens",
    "warm woolen mittens",
    "Cream-colored ponies",
    "crisp apple strudels"
  ]
}
```
